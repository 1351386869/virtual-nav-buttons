# Virtual Navigation Buttons

Android app that displays floating back/home/recent buttons on screen.

## Features
- Back button (←)
- Home button (⌂)  
- Recent apps button (☰)

## Building

Requires Android Studio or Android SDK.

1. Open project in Android Studio
2. Sync Gradle
3. Run on device/emulator (API 26+)

## Usage

1. Install the APK
2. Launch the app
3. Grant overlay permission when prompted
4. Tap "启动悬浮按键" to show the navigation bar
5. Tap buttons to simulate navigation actions

## Notes

- Key injection (back/recent) may require system signature or root
- Home button works without special permissions
- Works on Android 8.0+ (API 26+)
